## 第1节 — 活动前规划

### 1.1 组织批准
- [ ] 活动方案已提交 Committee 并获批准。
- [ ] 活动符合 HCCA 的慈善宗旨（文化遗产、社区支援、教育、文化融合）。
- [ ] 活动预算已编制并经 Committee 批准。
- [ ] 若本次活动将使用 Overseas Funding：已与 Treasurer 确认 Overseas Funding 合规要求。

### 1.2 场地
- [ ] 场地预订已确认，并有书面记录。
- [ ] 场地符合 Building Act 2004 要求（如适用，须展示有效的建筑合规证书）。
- [ ] 场地容量已确认，且不会超员。
- [ ] 无障碍需求已评估（轮椅通道、助听设备等）。
- [ ] 场地紧急出口已确认且畅通无阻。
- [ ] 已向场地管理方取得紧急疏散程序。
- [ ] 已向场地管理方确认急救箱位置。

### 1.3 法律及监管要求
- [ ] 已取得必要的许可证或执照（如使用公共场地须申请议会活动许可，提供食品须持有食品处理许可）。
- [ ] 若提供酒精饮料：已依据 Alcohol Act 向当地议会申请特别牌照。
- [ ] 若收集参与者个人信息：已遵守 Privacy Act 2020 的义务；已为参与者准备隐私声明。
- [ ] 若对参与者进行拍摄或录像：已准备同意书。
- [ ] 若活动涉及儿童：已落实儿童安全环境原则；相关志愿者持有有效的警察背景调查证明（见 Volunteer Policy）。

### 1.4 健康与安全 *(Health and Safety at Work Act 2015)*
- [ ] HCCA 已就适用情形履行 PCBU（Person Conducting a Business or Undertaking）的职责。
- [ ] 已就场地及活动内容完成危险识别。
- [ ] 已发现的危险已记录，并制定相应的缓解措施。
- [ ] 已准备紧急联系人名单（急救员、活动协调人、场地管理方、紧急服务电话）。
- [ ] 户外活动已制定恶劣天气应急预案。

### 1.5 财务
- [ ] 活动预算已经 Committee 批准。
- [ ] 票价（如适用）已经 Committee 批准。
- [ ] 收款方式已批准（现金处理规则、支付平台设置）。
- [ ] 已确认费用审批流程（谁可批准当天临时产生的费用及审批上限）。
- [ ] 零用现金（如需要）已经 Treasurer 批准。

---

## 第2节 — 活动进行中

### 2.1 场地开放前
- [ ] 活动协调人至少在活动开始前60分钟到达现场。
- [ ] 确认紧急出口畅通无阻。
- [ ] 确认急救箱位置。
- [ ] 已完成志愿者情况简介（职责分工、紧急处置程序、事故报告）。
- [ ] 已设立签到册（用于责任记录及紧急点名）。

### 2.2 活动期间
- [ ] 出席人数未超出场地容量。
- [ ] 任何事故、伤害或投诉已记录于下方的活动事故记录表。
- [ ] 若收集个人信息，隐私声明已张贴或分发。
- [ ] 任何计划外的临时费用已由现场指定审批人批准。

### 2.3 收入与现金管理
- [ ] 所有现金收入已由两人共同点算并核实。
- [ ] 现金收据已记录，并在 **2个工作日内** 存入 HCCA 银行账户。
- [ ] 电子付款记录已保存。

---

## 第3节 — 活动结束后

### 3.1 财务
- [ ] 活动后30天内完成财务摘要。
- [ ] 所有收据及发票已收集并存档。
- [ ] 活动收现已存入银行（活动结束后2个工作日内）。
- [ ] 已编制预算与实际对比分析表并提交 Treasurer。

### 3.2 文件归档
- [ ] 已记录实际出席人数。
- [ ] 本清单已完成并提交 Secretary 存档。
- [ ] 任何事故报告已存档。
- [ ] 志愿者服务时数已记录。
- [ ] 已收集参与者反馈（如适用）。

### 3.3 活动回顾
- [ ] 已与活动团队进行活动总结简会。
- [ ] 已记录经验教训，供未来活动参考。
- [ ] 公开发布照片／视频前已进行隐私合规审查。
